Easy Start will connect the printers and install printer drivers.

  1. Make sure your printer is turned on.

Welcome

Welcome to HP printers installation. Follow the instructions to get started.

Connecting Printers


USB Connection

Connect the Printer to this computer using a USB Cable.

Network Connection

Connect the printer to same network as this computer. If already connected, the installer will detect it.

Wireless Network Connection

Connect the printer to the same wireless network as this computer. If not connected, the installer will help you set it up.

Pairing Printers

  1. Turn on the printer
  2. Wait until the printer is ready
  3. Click the "Next" button
